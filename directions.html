NOTICE: This test may not be ideal for individuals who are sensitive to loud noises or sudden changes in noise.

DIRECTIONS

This test must be performed with a direct audio cable or Bluetooth connection. This ensures that the Frequency Response of the Cochlear Implant(s) remains accurate.

The next page contains the test interface. The test interface consists of three parts: calibration, adjustment, and testing.

On the testing page, you can name the test and set the pitch interval. The default is 200 Hz but you can set it between 1 and 1,000. The higher the value, the faster but less accurate the test will be.

Remember to use a useful name for the test such as a program name/number or date. The file name will end with the side you test (left, right, or both) and test date.
